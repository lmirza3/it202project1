About this Page

Data: I utilized the Chicago Data Portal. I made a call that returned the amount of farmers markets as well as another call that retrieves the amount of green roofs.

I used the example "Cover" available on the bootstrap getting started page (link here)

I utilized the Bootstrap framework because it was the most comfortable for me to use.

This project took me about 15-20 hours total. The most difficult part, for me, was just finding relevant data to utilize and dealing with different API's. After toying with the POKEAPI for a long time, I had a lot of difficulty because it doesn't have an aggregate list with all information, making the process of retrieving data for multiple pokemon difficult. I also attempted using Spotify's API, however, I planned to use it with categorical data and Spotify doesn't allow those who don't have an account to use genre data, making it hard for me to share the data with everyone. Eventually, I returned back to using mapping data because I know it's pretty reliable and complete. I intend to complete the pokedex later on.
